# AB-WRO-ANDR-ES-DE-VANDELVIRA
WRO PROJECT ANDRÉS DE VANDELVIRA

Para hacer el coche hemos utilizado piezas Nezha. Primero hemos hecho el chasis con barras abajo del todo para poder poner el módulo donde va la tarjeta microbit con motores y sensores de ultra sonidos. Hemos utilizado dos motores, uno con un sistema de giro en las ruedas de delante para poder girar y el otro motor lo hemos puesto atrás para que las ruedas traseras del coche puedan hacer la fuerza para avanzar y pudiendo girar con el motor y el sistema de giro. Luego hemos puesto una cámara HuskyLens que detecta colores de momento para los bloques de colores, el aparcamiento y las paredes. También tenemos dos sensores de ultrasonidos, un delante para detectar la distancia de las paredes de delante y saber cuando girar en las esquinas y otro en el lado derecho para que el coche se pueda guiar teniendo de referencia la pared de la derecha. 

Hemos puesto otro nuevo sensor de ultrasonidos en el lado izquierdo para poder calibrar mejor el como va el coche por el medio de la pista y también pata que empezmos por donde empezemos, dé igual y pueda avanzar  tranquilamente.
